# Create province names ENUM in a duckdb connection

Create province names ENUM in a duckdb connection

## Usage

``` r
spod_duckdb_create_province_enum(con)
```

## Arguments

- con:

  A `duckdb` connection.

## Value

A `duckdb` connection with `INE_PROV_NAME_ENUM` and `INE_PROV_CODE_ENUM`
created.
